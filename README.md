# FyloTech

## üìú Project Description

FyloTech is an **e-commerce website prototype** built around the concept of selling modern gadgets and technology-based products. It is initially developed as a static/front-end template to demonstrate the basic, yet professional, structure (header, footer, navigation) and design of a modern online store.

* **Project Aim:** To create a clean, well-organized, and user-friendly e-commerce interface.
* **Key Features:** Comprehensive navigation, dedicated sections for products, contact, and legal information.

## ‚öôÔ∏è Setup & Installation Instructions

Follow these steps to get the project running on your local machine.

### Prerequisites

* **Git** (To download the project code)
* **Node.js** (If your project uses a JavaScript framework like React, Next.js, or relies on Node scripts)
* **Modern Web Browser** (e.g., Chrome, Firefox, Edge)

### Installation Steps

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/ferdaws-ahmed/FyloTech-client.git]
    ```

2.  **Navigate to the Project Directory:**
    ```bash
    cd fylo-tech
    ```

3.  **Install Dependencies (If using Node.js):**
    ```bash
    npm install
    # or
    yarn install
    ```

4.  **Run the Project (Start Development Server):**
    ```bash
    npm run dev 
    # or 
    npm start
    ```
    After running this command, the project will usually start at `http://localhost:3000` (or another port). You can open this URL in your web browser.

## üó∫Ô∏è Route Summary

The website is structured with the necessary pages to simulate a complete e-commerce experience:

| Route / Path | Page Title / Description | Purpose |
| :--- | :--- | :--- |
| **`/`** | **Home** | The main landing page for the website. Displays banners and featured products. |
| **`/products`** | **Products** | The main listing area for all products. This page may include filtering and sorting options. |
| **`/about`** | **About Us** | Provides information about the company, its mission, and vision. |
| **`/contact`** | **Contact** | Contains a contact form and company contact details for user inquiries. |
| **`/faq`** | **FAQ** | Answers to frequently asked questions to assist users. |
| **`/login`** | **Login / Register** | Page for existing users to log in and new users to create an account. |
| **`/privacy`** | **Privacy Policy** | Details the website's privacy rules and how user data is handled. |
| **`/shipping`** | **Shipping Info** | Provides detailed information regarding product delivery and logistics. |
